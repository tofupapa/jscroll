jScroll v1.0.0

jScroll is a javascript class & jQuery plugin, based on jQuery 1.x

You can easily use it to scroll blocks in a HTML tag container. It compatible with mouse and touch devices

How to use jScroll

1. Run as javascript class

e.g. new jScroll('tag id', [v:vertical(default)]|h:horizontal], [width], [height]);

STEP 1

Add css style to your HTML page:

.jScroll {overflow:hidden;-moz-user-select:none;-webkit-user-select:none;user-select:none;}

STEP 2

Load jquery.js and jquery.jscroll.js

<script type="text/javascript" src="script/jquery-1.9.1.min.js"></script>
<script type="text/javascript" src="script/jquery.jscroll.js"></script>

STEP 3

Run it

<script type="text/javascript">new jScroll('#demo1');</script>
